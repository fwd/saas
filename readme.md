# fwd/database

> NPM package for bootstrapping NodeJS SaaS development. 

## Install

```sh
npm install fwd/saas
```

## Usage

```js

const database = require('@fwd/database')('local')

const saas = require('@fwd/saas')({
    database: database,
})

saas.start(8080)

```
